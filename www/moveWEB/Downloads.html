Download a version of the move package

Because we frequently upload changes to r-forge the code will be broken from time to time. To offer you a stable, yet not the most recent version of move we will upload here current stable versions of the move package.


However, check out our r-forge page to get the latest version.


Note: at the moment there is no move package available for Windows users. This is because we support internet based functions to browse the Movebank database from within R. This is currently based on the RCurl package, which is not available on CRAN as a binary. As soon we implemented an RCurl free version, there will be also a windows Version again.
