The Move package

 
 

With ,move‘ we created an R package that contains functions to access movement data stored in movebank.org as well as tools to visualize and statistically analyze animal movement data. Move is addressing movement ecological questions complementing existing efforts such as adeHabitat and other packages.

The R package for tracking and Movebank data

With ,move‘ you can easily visualize animal track data you recorded or data from the data base movebank.org. Plot the track as lines, points, or on a google map to get a better feeling for your data.

Visualize animal track data

With ,move‘ you can calculate the utilization density of your animal using the dynamic Brownian Bridge Movement Model (see Kranstauber, B., Kays, R., LaPoint, S. D., Wikelski, M. and Safi, K. (2012), A dynamic Brownian bridge movement model to estimate utilization distributions for heterogeneous animal movement. Journal of Animal Ecology. doi: 10.1111/j.1365-2656.2012.01955.x). Change the color of your plot and add the track and contour lines to it.
